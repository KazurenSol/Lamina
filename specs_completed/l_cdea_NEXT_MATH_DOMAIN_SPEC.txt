# CANONICALIZATION RULES

CAS MUST detect algebraic equivalence.

Examples:

a + b = b + a  (commutative)

(a + b) + c = a + (b + c)  (associative)

2 * (a + b) = 2a + 2b  (distributive)

x^2 + 2x + 1 = (x + 1)^2

a / b = a * b^-1

Equivalent expressions MUST normalize to the same canonical signature.

---

# SIMPLIFICATION RULES

SIMPLIFY MUST:

1. Reduce expression complexity
2. Preserve equivalence
3. Apply canonical ordering
4. Eliminate redundant operations

Example:

x + 0 → x  
x * 1 → x  
x * 0 → 0

# EXECUTION RULES

1. All math operators MUST be deterministic
2. No floating-point randomness in V1
3. Exact symbolic results preferred over approximations
4. Approximation must be explicit if used
