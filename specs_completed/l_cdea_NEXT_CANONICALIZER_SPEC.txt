CompiledOutput (multiple CDLGraphs + bindings + interpretations)

# HARD INVARIANTS

1. Canonicalizer MUST be pure (no mutation of input graphs)
2. Canonicalizer MUST preserve semantic equivalence classes

# 1. equivalence.py — GRAPH EQUIVALENCE ENGINE

PURPOSE:
Determine structural equivalence between CDLGraphs.

RULES:
Two graphs are equivalent if:
- operator sequences are isomorphic under canonical renaming
- TypeSignatures match exactly
- edge directionality preserved
- terminal values structurally identical

OUTPUT:
EquivalenceClassID

---

# 2. signature.py — CANONICAL GRAPH SIGNATURES

PURPOSE:
Convert CDLGraph → hashable structural signature

RULES:
- uses operator identity (from cached operator registry)
- uses type signatures only (no runtime values)
- ignores insertion order (must use topological normalization)
- must be deterministic across runs

OUTPUT:
CanonicalSignature (hashable)

---

# 3. normalize.py — GRAPH NORMALIZATION LAYER

PURPOSE:
Convert equivalent graphs into a canonical structural form.

RULES:
- reorder commutative operations (if defined)
- normalize argument ordering
- standardize operator chains
- flatten redundant intermediate nodes

OUTPUT:
NormalizedCDLGraph

---

# 4. merge.py — EQUIVALENCE COLLAPSE ENGINE

PURPOSE:
Merge structurally equivalent graphs into equivalence classes.

RULES:
- does NOT delete graphs
- creates equivalence groups
- preserves all interpretations as members of class

OUTPUT:
CanonicalGraphSet {
    equivalence_classes: Dict[CanonicalSignature, List[CDLGraph]]
}

---

# 5. exceptions.py

DEFINE:

class CanonicalizationError(Exception):

class EquivalenceDetectionError(CanonicalizationError):
    pass

class NormalizationError(CanonicalizationError):
    pass

# CRITICAL DESIGN RULES

No semantic interpretation allowed.

## 2. No MECP influence
Do not prune based on cost or usefulness.

## 3. No DiscourseState access
Memory is strictly forbidden here.

## 4. No execution of graphs
Graphs remain inert objects.

## 5. No loss of information
All original graphs must remain recoverable via equivalence mapping.
